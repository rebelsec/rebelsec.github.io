Demo: alert(), prompt(), confirm()

Tujuan: lihat perilaku omnibox ketika dialog muncul. JANGAN masukkan informasi sensitif saat menguji.

Tips tes:
  1. Pindahkan omnibox (address bar) ke bawah/atas di Chrome.
  2. Buka file ini di Chrome Stable.
  3. Klik tombol Alert/Prompt/Confirm dan perhatikan apakah omnibox terselubung/hilang sementara.
  4. Jangan memasukkan kredensial apapun — ini hanya demo UI.
