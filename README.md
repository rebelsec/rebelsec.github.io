## 🚀 Cara Berkontribusi

Kami terbuka untuk komunitas! Jika ingin berkontribusi dalam pengembangan Rebelsec:

1. **Fork repository ini**
2. **Clone repo ke lokal**
   ```sh
   git clone https://github.com/rebelsec/rebelsec.github.io.git
   ```
3. **Buat branch baru untuk fitur/bugfix**
   ```sh
   git checkout -b fitur-baru
   ```
4. **Commit perubahan**
   ```sh
   git commit -m "Menambahkan fitur baru"
   ```
5. **Push ke GitHub**
   ```sh
   git push origin fitur-baru
   ```
6. **Buat Pull Request** dan tunggu review dari tim Rebelsec.
